![Power BI](https://drive.google.com/file/d/1IVlw1mMZgi9g6E1iiX1VpfCXCC5b7KrV/view?usp=sharing)

## 📌 Project Overview
This project showcases an interactive **Financial Dashboard** developed in **Power BI** to monitor financial performance through revenue, profit, costs, budgeting, receivables, payables, and expenses. The dashboard is designed to support business users with data-driven decision making using interactive visuals, KPIs, and filters.

## 🎯 Business Problem
Organizations generate large volumes of financial data across departments and regions. Decision-makers need a centralized dashboard to monitor financial performance, compare actuals with budgets, track receivables and payables, and identify spending trends.

## 🎯 Objectives
- Monitor revenue, profit, and cost.
- Compare budget against actual performance.
- Analyze Accounts Receivable (AR) and Accounts Payable (AP).
- Track expense approvals and rejections.
- Provide interactive filtering by department, region, and time.

## 📂 Dataset Overview
A sample dataset is included in this repository for demonstration purposes. The original Power BI project file is available through Google Drive because it exceeds GitHub's file size limit.

## 🧹 Data Cleaning & Transformation
- Verified data types.
- Created relationships between fact and dimension tables.
- Built calculated measures using DAX.
- Organized tables for reporting and analysis.

## 🏗️ Data Model
The model follows a star-schema style with fact tables connected to supporting dimension tables such as customers, departments, regions, products, vendors, and dates.

## 📐 DAX Measures
| Measure | Purpose |
|---------|---------|
| Total Revenue | Total revenue generated |
| Total Profit | Sum of profit |
| Total Cost | Total operational cost |
| Total Budget | Budget allocation |
| Total Actual | Actual amount |
| Budget Variance | Actual - Budget |
| Profit Margin % | Profit ÷ Revenue |
| Total Customers | Distinct customer count |
| Avg Revenue Per Customer | Revenue per customer |
| Outstanding AR | Outstanding receivables |
| Outstanding AP | Outstanding payables |
| Total Expenses | Total expenses |
| Approved Expenses | Approved expense amount |
| Rejected Expenses | Rejected expense amount |
| Overdue % | Percentage of overdue payments |
